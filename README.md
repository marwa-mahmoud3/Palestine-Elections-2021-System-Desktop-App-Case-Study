## üéØ Project Purpose
The primary goal was to provide a robust digital platform for organizing the electoral workflow. It enables users to systematically create electoral blocks, register candidates with validated details, and track voting results to view calculated outcomes.

### **Main Functionalities:**
* **Electoral List Management**: Users can create, name, and assign unique identification numbers to various electoral lists.
* **Candidate Registration**: Detailed member entry for each list, including personal identification (ID), age, and gender.
* **Results Tracking**: A dedicated module to input vote counts per list, which then displays the total number of seats and percentages.
* **Data Overviews**: Comprehensive data grids to view all registered members and competing lists in a centralized location.
